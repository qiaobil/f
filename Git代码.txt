在你的项目文件夹 鼠标右键点击 git Bash Here

git init //创建空白仓库名(必填）
 
git add .  //获取全部文件（我一般用这个）

git add 123.txt（获取单个文件）

git add css （获取单个文件夹）

git commit -m "update index.html." // 文件描述（必填）

第三步：发送数据到gitee

git remote add origin https://gitee.com/用户名/路径.git

git push -u origin master （master=分支名）//发送数据到gitee

git push -f origin master （master=分支名）//强制发送数据到gitee（慎用）

输入完（git push -u origin master）可能会要求你输入邮箱+密码
